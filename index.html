Cara Ganti Password & Nama Wi-Fi

Pilih modem yang ingin Anda atur:

Cara Ganti Password & Nama Wi-Fi - ZTE

  1. Hubungkan perangkat ke Wi-Fi.
  2. Buka browser dan masukkan 192.168.1.1
  3. Login dengan:
    • Username: user
    • Password: user
  4. Pilih menu Local Network → WLAN → WLAN SSID Configuration.
  5. Ubah:
    • Name SSID: untuk mengganti nama Wi-Fi.
    • WPA Passphrase: untuk mengganti password.
  6. Klik Simpan atau Apply.

Cara Ganti Password & Nama Wi-Fi - FiberHome

  1. Hubungkan perangkat ke Wi-Fi.
  2. Buka browser dan masukkan 192.168.1.1
  3. Login dengan:
    • Username: user
    • Password: user
  4. Buka menu Network → 2.4G Advanced atau 5G Advanced.
  5. Ubah:
    • Name SSID: untuk mengganti nama Wi-Fi.
    • WPA Passphrase: untuk mengganti password.
  6. Klik Apply.

Cara Ganti Password & Nama Wi-Fi - Huawei

  1. Hubungkan perangkat ke Wi-Fi.
  2. Buka browser dan masukkan 192.168.100.1
  3. Login dengan:
    • Username: Admin
    • Password: admin
  4. Pilih menu Network → WLAN Settings → Advanced.
  5. Ubah:
    • SSID Name: untuk mengganti nama Wi-Fi.
    • Passphrase: untuk mengganti password.
  6. Klik Apply.
