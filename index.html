Cara Ganti Password & Nama Wi-Fi pada Modem

Cara Ganti Password & Nama Wi-Fi pada Modem ZTE

  1. Pastikan perangkat Anda sudah terhubung ke jaringan Wi-Fi.
  2. Buka Google Chrome (atau browser lain yang Anda pilih).
  3. Ketik 192.168.1.1 pada bilah alamat browser dan tekan Enter. Ini adalah alamat IP untuk mengakses halaman login modem.
  4. Anda akan diminta untuk memasukkan Username dan Password:
    • Username: user
    • Password: user
  5. Setelah berhasil login, pilih menu Local Network (terletak di bagian atas).
  6. Pada menu sebelah kiri, pilih WLAN.
  7. Scroll ke bawah dan pilih WLAN SSID Configuration.
  8. Di bagian ini, Anda dapat mengubah:
    • Name SSID: Untuk mengganti nama Wi-Fi sesuai keinginan Anda.
    • WPA Passphrase: Untuk mengganti kata sandi Wi-Fi.
  9. Setelah selesai mengubah, klik Simpan atau Apply untuk menyimpan perubahan Anda.

Cara Ganti Password & Nama Wi-Fi pada Modem FiberHome

  1. Pastikan perangkat Anda terhubung ke jaringan Wi-Fi terlebih dahulu.
  2. Buka Google Chrome (atau browser lain yang Anda pilih).
  3. Ketik 192.168.1.1 pada bilah alamat browser dan tekan Enter untuk membuka halaman login administrator modem.
  4. Anda akan diminta untuk memasukkan Username dan Password:
    • Username: user
    • Password: user1234
  5. Setelah login berhasil, pilih menu Network yang ada di bagian atas.
  6. Pilih salah satu dari dua opsi berikut, tergantung pada jaringan yang Anda gunakan:
    • 2.4G Advanced (untuk Wi-Fi 4G)
    • 5G Advanced (untuk Wi-Fi 5G)
  7. Pada halaman konfigurasi, ubah:
    • Name SSID: Ganti nama Wi-Fi sesuai keinginan Anda.
    • WPA Passphrase: Ganti kata sandi Wi-Fi Anda.
  8. Klik Simpan atau Apply untuk menyimpan perubahan.
